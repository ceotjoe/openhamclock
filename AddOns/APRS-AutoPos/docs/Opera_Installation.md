# Installation Guide for Opera

To use the **APRS Auto-Position** in Opera, follow these steps:

1. **Install Tampermonkey**:
   - Go to the [Opera Add-ons Store](https://addons.opera.com/en/extensions/details/tampermonkey-beta/).
   - Click **Add to Opera**.

2. **Add the Script**:
   - Copy the script content from [aprs_autopos.user.js](../aprs_autopos.user.js).
   - Open the Tampermonkey Dashboard and click the **+ (Plus)** icon to add a new script.
   - Paste the code and save.
