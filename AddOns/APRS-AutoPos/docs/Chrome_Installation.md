# Installation Guide for Google Chrome

To use the **APRS Auto-Position** in Google Chrome, follow these steps:

1. **Install Tampermonkey**:
   - Go to the [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo).
   - Click **Add to Chrome**.

2. **Add the Script**:
   - Open the [aprs_autopos.user.js](../aprs_autopos.user.js) file or copy its content.
   - Click the Tampermonkey icon and select **Create a new script...**.
   - Delete any placeholder code, paste the script, and go to **File > Save**.

3. **Enable Developer Mode (if required)**:
   - If the script doesn't run, go to `chrome://extensions/` and ensure **Developer mode** is toggled ON at the top right.

4. **Usage**:
   - Refresh your OpenHamClock page. Use the üìç icon at the bottom right to open the calculator.
