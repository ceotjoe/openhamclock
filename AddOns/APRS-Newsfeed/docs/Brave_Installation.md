# Installation Guide for Brave

To use the **APRS Newsfeed** in Brave, follow these steps:

1. **Install Tampermonkey**:
   - Brave is based on Chromium, so visit the [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo).
   - Click **Add to Brave**.

2. **Add the Script**:
   - Copy the content of [aprs_newsfeed.user.js](../aprs_newsfeed.user.js).
   - Click the Tampermonkey extension icon and select **Create a new script...**.
   - Paste the code and save via **File > Save**.

3. **Usage**:
